# Tokenized Community Garden Coordination Network

A decentralized platform for managing community gardens through blockchain technology, enabling transparent plot allocation, resource sharing, harvest distribution, educational coordination, and sustainability tracking.

## Overview

This network consists of five interconnected smart contracts that manage different aspects of community gardening:

### Contracts

1. **Plot Allocation Contract** (`plot-allocation.clar`)
    - Manages individual garden space assignments
    - Handles plot ownership and transfers
    - Tracks plot status and usage

2. **Resource Sharing Contract** (`resource-sharing.clar`)
    - Coordinates tool and equipment usage
    - Manages resource reservations
    - Tracks resource availability and maintenance

3. **Harvest Distribution Contract** (`harvest-distribution.clar`)
    - Manages produce sharing among participants
    - Handles harvest recording and distribution
    - Tracks contribution and sharing ratios

4. **Educational Workshop Contract** (`educational-workshop.clar`)
    - Organizes gardening skill development sessions
    - Manages workshop scheduling and attendance
    - Tracks learning progress and certifications

5. **Sustainability Tracking Contract** (`sustainability-tracking.clar`)
    - Monitors environmental impact and benefits
    - Tracks carbon sequestration and water usage
    - Records biodiversity and soil health metrics

## Getting Started

### Prerequisites
- Stacks blockchain environment
- Clarity development tools
- Node.js for testing

### Installation

1. Clone the repository
2. Install dependencies: \`npm install\`
3. Run tests: \`npm test\`
4. Deploy contracts to Stacks testnet

### Usage

Each contract can be deployed independently and provides specific functionality for community garden management. Refer to individual contract documentation for detailed usage instructions.
